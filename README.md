# Enterprise Data & AI Architectures — Exercises

Welcome to the repository for the exercises of the master's course **Enterprise Data & AI Architectures**. 

This repository is organized by modules, each containing the relevant exercises, hands-on tasks, and documentation.

## Course Modules

Below is the list of modules covered in this course. Click on any module link to view its description and corresponding exercises.

*   [**Module 1: Introduction to Software Architecture & The Evolution of Data Architectures**](./module-1/README.md)
    *   Foundations of software architecture and historical/conceptual evolution of data architectures.
*   [**Module 2: Data Lifecycle, Ingestion, and Processing**](./module-2/README.md)
    *   Techniques, pipelines, and frameworks for data lifecycle stages, ingestion strategies, and processing patterns.
*   [**Module 3: Data-Centric Architectures**](./module-3/README.md)
    *   Design patterns and structures optimized for data assets, storage systems, and integration.
*   [**Module 4: Cloud Infrastructure & Modern Data Platforms**](./module-4/README.md)
    *   Leveraging cloud environments, managed services, and modern cloud data warehouses/lakehouses.
*   [**Module 5: Advanced AI Architectures**](./module-5/README.md)
    *   Architectural patterns for scale, MLOps, LLMs, and agentic AI systems.
*   [**Module 6: Data Governance, Quality, and Security**](./module-6/README.md)
    *   Ensuring compliance, auditability, data privacy, security controls, and data quality standards.
*   [**Module 7: Artificial Intelligence (AI) Governance**](./module-7/README.md)
    *   Ethical frameworks, policy enforcement, safety guidelines, and monitoring in enterprise AI deployments.
